# Анализатор логов

Консольное приложение на Python для анализа логов и формирования HTML-отчёта по инцидентам.

## Возможности

* Чтение всех `.log` файлов из указанной директории.
* Парсинг логов в формате:

```text
2026-06-22 15:20:14 INFO User login success
```
* Игнорирование некорректных строк без остановки выполнения программы.
* Подсчёт общей статистики по уровням логирования:

  * INFO
  * WARNING
  * ERROR
  * CRITICAL
  * TIMEOUT
* Подсчёт количества инцидентов.
* Формирование ТОП-10 наиболее частых проблемных сообщений.
* Сбор статистики по каждому файлу логов.
* Генерация HTML-отчёта.

## Установка

Создать виртуальное окружение:

```bash
python -m venv venv
```

Активировать виртуальное окружение:

### Linux

```bash
source venv/bin/activate
```


Установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

Пример запуска:

```bash
python main.py --logs ./logs --output report.html
```

Параметры:

| Параметр   | Описание                   |
| ---------- | -------------------------- |
| `--logs`   | Путь к директории с логами |
| `--output` | Имя итогового HTML-отчёта  |



---

## Формат логов

Поддерживаемый формат строки:

```text
YYYY-MM-DD HH:MM:SS LEVEL Message
```

Пример:

```text
2026-06-22 15:20:14 INFO Application started
2026-06-22 15:21:01 ERROR Database connection failed
2026-06-22 15:22:30 CRITICAL Payment service unavailable
2026-06-22 15:23:01 TIMEOUT Request timeout
```

Некорректные строки автоматически фиксируются в отчёте и не приводят к остановке обработки.

## Формируемый отчёт

Отчёт содержит:

### Общая сводка

* Общее количество событий
* Количество INFO
* Количество WARNING
* Количество ERROR
* Количество CRITICAL
* Количество TIMEOUT

### Инциденты

Количество проблемных событий:

```text
ERROR + CRITICAL + TIMEOUT
```

### ТОП ошибок

Список из 10 наиболее часто встречающихся проблемных сообщений.

### Статистика по файлам

Для каждого файла отображается:

* Общее количество записей
* INFO
* WARNING
* ERROR
* CRITICAL
* TIMEOUT

### Некорректные строки

Список строк, которые не удалось распарсить, с указанием файла и номера строки.
